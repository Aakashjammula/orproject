## Usage
Make sure you have python installed

1. Clone repo
  ```sh
    git clone https://github.com/AlexKressner/Vehicle_Routing_Problem
  ```
2. Change directory
  ```sh
    cd Vehicle_Routing_Problem
  ```
3. run app.py
  ```sh
    python app.py
  ```
4. click on these link to see the website ("http://127.0.0.1:5000")
